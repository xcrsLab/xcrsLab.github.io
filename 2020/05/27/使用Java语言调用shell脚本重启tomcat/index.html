Java语言调用shell脚本

shell脚本

包含重启tomcat的指令，需要拷贝到Linux下，并将该文件后缀命名为”.sh”，拷贝到Linux系统后，使用vi编辑器将该文件打开，使用”set ff”命令查看当前的读取方式

如果fileformat=dos，将其修改为unix，如图，保存并退出。

使用 “chmod 777”命令给”.sh”文件赋权限。

调用java方法运行shell脚本即可完成tomcat的重启
